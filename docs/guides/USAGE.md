# 像素桌面桌宠 v0.2 使用文档

> 适用范围：当前仓库内的 v0.2 骨架实现（悬浮 + 拖拽 + 点击反馈 + 吸附 + 简单走动）。

## 1. 运行前准备

1. 默认资源为 `app/src/main/res/drawable/normal.xml` 与 `app/src/main/res/drawable/tongue.xml`，可替换为同名自定义素材。
2. 运行 App 后，在系统设置中授予 **悬浮窗权限**。
3. 预设素材包会在打包时自动生成 PNG，无需手动执行命令。

## 2. 基本使用流程

1. 打开 App。
2. 点击 **“启动桌宠”**。
3. 桌宠会显示在屏幕上层，可拖动移动。
4. 松手后会自动吸附到左右边缘。
5. 点击桌宠，会触发吐舌头反馈（500ms 后恢复）。
6. 桌宠会以随机节奏在边缘带内小步移动。
7. 点击 **“停止桌宠”** 可隐藏。

## 3. 常见问题

- **桌宠没有显示**：请确认已授予悬浮窗权限。
- **无法拖动**：请确认 App 未被系统限制后台权限。
